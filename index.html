The Learning Style Inventory Assessment

Use the following instructions or your intuition to complete the table.

  • Assign a number to each choice across the row.
  • The number you pick will be between 1 and 4.
  • You can only use each number once for each row.
  • Give the higher number to the phrase that you like more.
  • Click Select or white colored numbers in table to make a selection. You can always make a change.
  • Click Done when you have finished with the table.
